## 🚀 Features
- User can add data in todo app
- User can delete data from todo app
- User can edit data from todo app
- Responsiveness for all screen sizes

## Run Locally :
Clone the Project
```
https://github.com/kundan799/Captain_Marvel
``` 

Install Dependencies
```
npm install
```

Run App
```
npm run dev
```
